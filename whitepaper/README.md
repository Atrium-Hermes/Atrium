# Atrium Whitepaper

Source for *"Atrium: A Skill Provenance and Royalty Marketplace for AI Agents"*.
Markdown chapters + Mermaid diagrams, built to PDF with pandoc (LaTeX backend).

## Layout

```
whitepaper/
├── chapters/        # one Markdown file per chapter, concatenated in filename order
│   ├── 00-frontmatter.md … 09-future.md, 99-references.md
├── diagrams/        # Mermaid sources (.mmd) → rendered to build/*.pdf
├── metadata.yaml    # pandoc document metadata (title, abstract, geometry, TOC, bib)
├── refs.bib         # BibTeX bibliography (cited via [@key], rendered by citeproc)
├── build.sh         # render diagrams → concat → pandoc → atrium-whitepaper.pdf
├── Makefile         # `make pdf`, `make clean`, `make diagrams`
└── README.md
```

Chapters are concatenated in lexical filename order, so the numeric prefixes
(`00`–`99`) define document order. Diagrams are referenced from chapters by
basename (e.g. `architecture.pdf`); `build.sh` renders each `.mmd` into `build/`
and passes that on pandoc's `--resource-path`.

## Build

Requires **pandoc**, a **LaTeX engine** (`xelatex`, `pdflatex`, or `tectonic`),
and the **Mermaid CLI** (`mmdc`) for diagrams.

```bash
# macOS:  brew install pandoc mermaid-cli && brew install --cask mactex-no-gui
# Debian: apt-get install pandoc texlive-xetex && npm i -g @mermaid-js/mermaid-cli

./build.sh                 # → atrium-whitepaper.pdf
make pdf                   # same
PDF_ENGINE=tectonic ./build.sh   # override the LaTeX engine
make clean                 # remove build/ and the PDF
```

If `mmdc` is absent the build still produces a PDF, with diagram placeholders
missing; install the Mermaid CLI for the full document.

## Editing

- Keep each chapter independently readable; cross-reference by chapter number.
- Cite sources with `[@key]` using keys from `refs.bib`; the reference list is
  generated by `--citeproc` under the *References* heading.
- Edit diagrams in `diagrams/*.mmd` and re-run `make diagrams` to preview.
- Style: third person, concrete numbers, tables/diagrams over prose for structure,
  no marketing language. Numerical claims about Atrium follow the v0.1 contract.
