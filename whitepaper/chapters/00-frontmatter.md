## Status of this document {-}

This is a technical specification and design rationale for Atrium, version 0.1. It
describes a system that is implemented as a minimum viable product: a deployed
contract design, a reference command-line client, a Model Context Protocol
server, an event indexer, and a web interface. Numerical parameters (fees,
royalty caps, gas figures) reflect the v0.1 contract and are subject to the
governance process described in Chapter 7. Where this document makes empirical
claims about external systems, they are cited; where it describes Atrium's own
behavior, the smart contract source is the authoritative reference.
